# SOPS File Data Source

The SOPS file datasource provides access to SOPS encrypted secret files.

Basic example of usage:

```hcl
data "sops-file" "basic-example" {
  file = "test.yml"
  format = "yaml"
}

# usage example of the data source output
locals {
  value         = data.sops-file.basic-example.decrypted
}
```
