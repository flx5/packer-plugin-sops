# SOPS Content Data Source

The SOPS content data source allows decrypting SOPS encrypted content.

Basic example of usage:

```hcl
data "sops-content" "basic-example" {
  content = file("encrypted.yml")
  format = "yaml"
}

# usage example of the data source output
locals {
  value         = data.sops-content.basic-example.decrypted
}
```
